# Proyecto de Predicción de Pesos y Alturas

Este proyecto utiliza Machine Learning para predecir el peso de una persona basándose en su altura. 
Es el trabajo final para el curso de **Machine Learning II**.

## Contenido del Proyecto
- **Análisis Exploratorio (EDA):** Limpieza de datos y visualización estadística.
- **Ingeniería de Datos:** Creación de la variable IMC.
- **Modelado:** Comparación entre Regresión Lineal, KNN y Árboles de Decisión.

## Tecnologías utilizadas
- Python (Pandas, Scikit-learn, Matplotlib)
- Jupyter Notebooks / Google Colab
